Usage instruction
// ENG
To use the program, you need to copy the grammar.txt file (it contains the grammar of the programming language) to the executable directory.
The example.txt text file contains an example program in a given programming language.
Input values are input and output via the console.

// UA
Для використання програми необхідно скопіювати файл grammar.txt (він містить граматику мови програмування) в каталог виконуваного файлу.
Текстовий файл example.txt містить приклад програми на заданій мові програмування. 
Введення вхідних значень та вивід результатів здійснюється через консоль.
